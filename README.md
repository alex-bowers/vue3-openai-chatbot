# OpenAI Chat Bot

A Vue 3/Node app to show how a customer could use the technology.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Run Node

```sh
cd chatgpt-backend
node index.js
```
